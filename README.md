## üß† Concepts Covered

### Confidence Interval
A confidence interval provides a range of values that is likely to contain an unknown population parameter, such as a mean or proportion, with a specified level of confidence (e.g., 95%).

### Hypothesis Testing
Hypothesis testing is a statistical method used to decide whether there is enough evidence in a sample to support or reject a claim about a population.

Typical steps include:
1. Defining null and alternative hypotheses  
2. Choosing a significance level (Œ±)  
3. Computing a test statistic and p-value  
4. Making a decision based on the results
